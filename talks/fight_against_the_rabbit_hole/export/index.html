Ticket Refinement

This would become one of your best friends

Ask for acceptance criteras that can be used to

  • Create a test
  • Understand if the ticket if complete

Ask for dev resoruces

  • This belong to… what micro service?
  • This belong to… what function, or class?
  • Do we have PR’s with similar things?
  • Do this feature have a flow diagram?
  • Do this feature have a document?
  • Do this feature have a test plan?
  • What are the upstream service impacted?
  • What are the downstream services involved?
